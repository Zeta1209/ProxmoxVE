# Project Zomboid Proxmox LXC Installer

This script automatically creates a Debian 13 LXC on Proxmox and installs a Project Zomboid dedicated server with a web-based management interface.

---

## Installation

Run **on the Proxmox node shell**:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Zeta1209/ProxmoxVE/main/project-zomboid/pzserver.sh)
